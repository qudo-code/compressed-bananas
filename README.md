## Start Dev Server

-   `npm i`
-   `npm run dev`

### Mint Route (`/mint`)

-   Generates TipLink wallet.
-   Mints NFT to TipLink wallet.

### Included Setup Routes (`/setup/*`)

There are a couple routes in `/setup` that are handy when initially creating your collection account and merkle tree.
